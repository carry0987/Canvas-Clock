# Canvas-clock
**A simple canvas clock**

## Usage
You can change color in **[time.js](./js/time.js#L4-L6)** file :<br />
```
var circlecolor = '#00ffff'; //Circle color
```
```
var fontcolor = '#00ffff'; //Font color
```
```
var backgroundcolor = '#03303a'; //Background shadow color
```
